Todo arquivo no seu computador possui o que chamamos de extensão do arquivo, por exemplo, arquivos de imagem tem a extensão: .jpg, .jpeg. Da mesma forma arquivos html também possuem sua extensão, que é: .html

ficando algo parecido com isso: index.html

A marcação(Markup) do html, se dá pela estrutura: <nomeDaTag> conteúdo da tag/elemento </nomeDaTag>. Ex: <h1> marcação que define o título principal da página </h1>.

No html existem elementos vazios, que são elemento que não possuem a tag de fechamento: </nomeDaTag>, só possui a tag de abertura, que é justamente a tag que fica a frente do conteúdo. Ex: <img> é a tag que serve para adicionar imagens ao arquivo.

Elementos html possuem atributos, que são a forma de adicionarmos configurações e informações extras ao nosso elemento html, já que ele por sí só não faz muita coisa. Por exemplo, a tag para adicionar imagens, precisa de dois atributos para poder funcionar de fato.

A sintaxe de todo atributo é: nomedoatributo="valordoatributo"

todo atributo vai dentro da tag de abertura, sendo separados por um espaço, ex: <img src="caminho do arquivo de imagem" alt="texto alternativo que vai parecer se a imagem não funcionar" >

O atributo: src="" serve para adicionarmos o link de uma imagem para aparecer no nosso site, ele é obrigatório para a tag img poder funcionar, já o atributo: alt="" é para adicionarmos um texto alternativo caso a imagem não apareça, e é um atributo muito importante para questões de acessibilidade, então sempre utilizar e especificar bem o que está na imagem através desse atributo, ex: alt="imagem de uma mulher segurando um guarda-chuva enquanto está parada olhando o por do sol".

Há também os atributos booleanos, que não precisam de conteúdo, booleano significando que apenas possuem dois tipos de valores, sendo eles verdadeiro ou falso, não havendo texto, apenas essas duas opções. <input type="text" disabled>

Outro fator que devemos levar em consideração são as aspas, recomendado apenas o uso das aspas duplas, para não desencadear problemas no código.

Existe também os atributos globais, que são basicamente atributos que servem em toooda tag html, algumas delas: class, id, title, style, entre várias outras.

O atributo: class, serve para quando formos querer manipular várias tags html da mesma forma, ou seja, poder por exemplo, fazer com que toda tag html que tenha o atributo class com o mesmo nome de valor, possua a mesma cor de texto. Ex: <input class="muda-cor-de-fundo" >, <p class="muda-cor-de-fundo" >, <img class="muda-cor-de-fundo" >.

O atributo: id serve para definir um estilo especifico para uma única tag html, não podendo existir outro id com o mesmo valor, esse atributo é muito utilizado para manipularmos o elemento html via linguagem de programação, como a JavaScript.

No html, podemos ter tags, dentro de outras tags, e isso se chama: aninhamento de tags, dessa forma podemos criar estruturas html ainda mais complexas, por exemplo, dentro da minha tag p, que serve para marcarmos um parágrafo, eu queira que uma das palavras fique maior e com cor diferente, eu poderia fazer aninhamento de tags aqui utilizando a tag: <span> que possui tag de fechamento, para poder fazer esse tipo de manipulação.

Exemplo de aninhamento de tag: <p> aqui é um parágrafo e <span>está</span> palavra quero mudar sua cor. <p>.

A tag span serve justamente para manipularmos pequenos textos dentro de um dado contexto.

Na contrução do html, vamos notar que o FLUXO de contrução da nossa página é toda de cima para baixo.

Outra coisa a se notar é que, existe um posicionamento dos elementos html, por exemplo, algumas tags ficaram em na mesma linha sem acabar quebrando a linha, como o caso ta tag span, que não quebrou a linha do parágrafo, mas também, existem tags que quebram a linha de exibição, por exemplo a própria tag p.

Caracteres reservado são caracteres usados no próprio HTML, como < > & " " ' ', não podemos usar nas tags, pois dão erro, mesmo aparecendo na preview o navegador vai tentar ler esses caracteres de alguma forma que faça sentido. Para usarmos precisamos troca-los por outras formas de escrever:
maior que: <
menor que: >
Aspas: " que colocamos "
Aspas simples: ' conteúdo '
existem várias outras

a estrutura base de todo documento html é: 

                    <!DOCTYPE html>
                    <html>
                        <head>
                            <meta charset="utf-8">
                            <title>Anatomia do Documento
                        </head>

                        <body>
                            aqui vai todo o conteúdo que será exibido na tela
                        </body>
                    </html>

O padrão seria esta forma.
<!DOCTYPE html> — diz ao navegador que estamos a trabalhar com HTML 5.
<html></html> — o próprio HTML, elemento raiz, o inicio da cadeia.
<head></head> — contém configurações importantes para página, mas não ainda o que o usuário vai ver.
<meta> — onde vai representar vários tipos de metadados da página.
<title> — título da página.
<body> — onde haverá conteúdo visual da página.

algumas regras para a contrução de nomes dos arquivos html: não pode conter espaços em branco no nome, tudo tem se quer junto ou separado por: -

não podemos utilizar caractes especias nos nomes, como: +, ^, *, $, entre outros, somente: - e o sinal de _

Ex: index.html, sessao-de-fotos.html, sessao_de_fotos.html
